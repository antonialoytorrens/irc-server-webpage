Welcome to my IRC Server!

IRC stands for Internet Relay Chat. It enables users to connect to a server using a software program or web service and communicate with each other live.

Here are a few guides to get you started:

  • Computer Hope - IRC

  • Mibbit Wiki

  • Just search the Internet.

Available Ports

There are two ports that let you connect to this service:

  • Port 6667 (not encrypted, plain text)
  • Port 6697 (encrypted by OpenSSL)

This site is only informative. If you see this page means that the server is running.
